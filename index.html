About the Est. KBB Delta & Value Rating

Each listing's estimated delta compares its asking price to a Kelley Blue Book Fair Purchase Price anchor for that model year/trim in typical condition. These are Claude-generated estimates from model-year valuation research, not VIN-specific KBB pulls — equipment, condition, and options can shift real book value by thousands. Confirm at kbb.com before making an offer. The Value Rating (Great Deal → Overpriced) is computed directly from that delta using fixed thresholds, not pulled from the listing site — so it's consistent across every result, unlike site-specific "deal ratings" that compare against local comps instead of book value.

About the data

Results are hand-researched snapshots compiled on the date shown, not a live universal search or scraper — every snapshot is produced by a Claude Chat search (using WebFetch/WebSearch, which get through the 403 blocks that plain scripted requests hit on sites like Cars.com and KBB). The default view (Jeep Wrangler, within 2 hours of 23185) matches the example report this app was built from. Searching a make/model/radius combination with no saved snapshot will show you the exact prompt to run to generate one.

About the "Link" column

These are Cars.com search links filtered to match each row's year/price/mileage/ZIP — not a guaranteed link to that exact listing. The original individual listing page URLs weren't preserved when this snapshot was compiled, so rather than guess a specific link that might 404 or point to the wrong vehicle, we link to the closest matching live search instead.

About the photos

Thumbnails are representative stock photos of that model's generation (e.g. JL-era Wrangler, 2018–present), not a photo of the specific vehicle listed — we don't have the original listing photos. About the inspection guidance

The Top 5 inspection panel is generic guidance, not a shop directory — we don't have real-time data on individual inspectors' pricing or availability, so we won't invent shop names. Cost ranges are typical U.S. market rates for an independent pre-purchase inspection (PPI).
